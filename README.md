# Daily Sangalpam

A simple Tamil daily sankalpam website.

## Usage

- Open `index.html` in a browser.
- The page shows a daily sankalpam in Tamil.
- Click "இன்னொரு சங்கல்பம்" to see another encouraging sankalpam.

## Files

- `index.html` — main page structure.
- `styles.css` — styling for the site.
- `script.js` — daily sankalpam logic.
